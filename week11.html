Modules



Modules are actually very simple once you understand them. All it essentially is, is using javascript functions in other files by import/exporting them. This is useful because that way you don't have to retype out all the properties of a function. These modules are very flexible because you can rename them as well as choose which functions transfer and which ones don't.

Higher Order Functions



Much like modules this is a very straight forward concept. As you can see in the two example picture above: both involve calling a function that has multiple uses so that you don't have to rewrite the same function over and over again with small differences. This is important because using it is going to save you a lot of time and headache. With higher order functions you will be able to finish tedious, redundant task a lot easier.

Advanced Array Methods

Advanced Array Methods allow arrow functions to be used with arrays.

  • find - returns whatever value in the array that satisfies the function requirements first

    • const array = [2, 4, 6, 8, 10];
    const num = array.find(element => element > 7);
    console.log(num);

    Output: 8

  • findIndex - much like 'find' this returns the index of the number in an array that meets the requirements

    • const array = [5, 12, 9, 130, 44];
    const isLargeNumber = (element) => element > 13;
    console.log(array.findIndex(isLargeNumber));

    Output: 3

  • flat - creates a new array that concatenates the sub-array elements

    • const array = [1, 2, [3, 4]];
    array.flat();

    Output: [1, 2, 3, 4]

  • map and flatMap - maps all elements then flattens the results into an array

    • let array = [1, 2, 3, 4];
    array.map(x => [x * 2]);

    Output: [[2], [4], [6], [8]]

    array.flatMap(x => [[x * 2]]);

    Output: [[2], [4], [6], [8]]

  • filter - creats a new array with only the elements that meet the requirements

    • const words = ['ball', 'hit', 'run', 'base', 'pitch', 'strike'];
    const result = words.filter(word => word.length > 4);
    console.log(result);

    Output: ['pitch', 'strike']

  • push - adds elements to an array then returns the length

    • const sports = ['baseball', 'hockey', 'soccer'];
    const count = sports.push('basketball');
    console.log(count , sports);

    Output: 4 ['baseball', 'hockey', 'soccer', 'basketball']

  • sort - sorts the elements inside an array in ascending order whether its numerical or alphabetical

    • const fruit = ['grape', 'apple', 'orange'];
    fruit.sort();
    console.log(fruit);

    Output: ['apple', 'grape', 'orange']

    const array = [ 5, 49 , 29 , 10];
    array.sort();
    console.log(array);

    Output: [ 5, 10, 29, 49]
