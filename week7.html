Prototypal Inheritance

Attributes and Functions

Every functions has a property called "prototype" by default. The property starts off as empty but is able to have properties and methods added to it. Creating an object from a function will cause the function to inherit its properties and methods.

`This` Keyword

`this` is a very versatile keyword that is always used to refer to an object.

Window Object

A window object refers to whichever window you have open in your browser. Everything that is global is inside the window object including variables, objects, and functions. Global variables are properties and global functions are methods of the window object.

Global Object

Global object relates to the `this` keyword because when used it refers to the global object regardless of whether or not the global execution context is in strict mode.
